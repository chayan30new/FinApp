# Watchlist - Stock Search Feature

## What's New
The watchlist now has a built-in stock/ETF search feature to help you quickly find and add Australian stocks and ETFs.

## How to Use

### Method 1: Type the Symbol Directly
1. Click "Add to Watchlist"
2. Type the ticker symbol (e.g., `VAS.AX`)
3. Name auto-fills with the symbol
4. Click "Add"

### Method 2: Search from the Database
1. Click "Add to Watchlist"
2. Click the **üîç Search icon** in the dialog title
3. Type to search by:
   - **Symbol** (e.g., type "VAS" to find VAS.AX)
   - **Name** (e.g., type "Vanguard" to find Vanguard ETFs)
4. Click on the stock you want
5. Symbol and name are automatically filled in
6. Click "Add"

## Tips
- The search is smart - it finds matches in both symbols and company names
- Type just a few letters to narrow down the list
- All symbols automatically include the .AX suffix for ASX stocks
- You can still manually type any symbol not in the database
